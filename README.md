# Hollow

Still under development

## Controls

| Input | Action |
|-------|--------|
| W / A / S / D | Move camera forward / left / backward / right |
| Mouse drag | Rotate camera / select objects |
| Arrow keys | Translate selected object |
| R | Rotate selected object |
| T | Scale selected object |
| F | Toggle fullscreen |
